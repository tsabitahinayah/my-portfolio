# Tsabitah's Portfolio

A modern, responsive personal portfolio website built with Next.js and Tailwind CSS, showcasing my skills, projects, and professional experience.

## Features

- **Modern UI Design** - Clean, professional interface with charcoal/emerald color scheme
- **Responsive Layout** - Optimized for all device sizes from mobile to desktop
- **Interactive Components** - Animated sections, carousels, and hover effects
- **Dark/Light Mode** - Automatic theme switching based on section
- **Performance Optimized** - Fast loading times and smooth animations

## Key Sections

- **Hero** - Introduction with animated typing effect
- **About** - Personal background and professional summary
- **Projects** - Showcase of completed work with image carousels
- **Experience** - Timeline of professional experience and achievements
- **Skills** - Technical skills and competencies
- **Certifications** - Professional certifications and training
- **Contact** - Contact form and social media links

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: CSS animations and React hooks
- **Deployment**: Vercel

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/tsabitahinayah/my-portfolio.git
cd my-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

The portfolio is designed to be easily customizable:

- Update personal information in component files
- Modify the color scheme in `tailwind.config.js`
- Add new projects to the `Projects.jsx` component
- Update experience in `OrganisationalExperience.jsx`

## Deployment

This portfolio is optimized for deployment on Vercel:

```bash
npm run build
```
